SimpleCanvas (scrapped?)

    A simple web painting application that uses the HTML5 canvas element. A lot of these types of apps currently exist around the internet. I mostly started this experiment to learn how to use the HTML5 canvas, although with some additional challenges:

    1. Add undo/redo functionality. At the time I started this, there did not seem to be any easily-findable painting apps that had this.
    2. Use only one canvas element for the painting/image surface (allowed to use additional canvas elements for other comonents, like the color-picker). Most canvas painting apps I found at the time I started used multiple canvas elements for the main painting component.

    I'm currently not working on this due to lack of time. Unfortunately, I don't have a working version at the moment either, as it is now broken and I had not bothered to version-control it at the time (sort of regret that now). Maybe I'll pick it up again in the future...
